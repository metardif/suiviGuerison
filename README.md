# 📊 Suivi Guérison L5/S1

Votre application personnelle de suivi de progression pour votre guérison.

## 🚀 Comment utiliser (sur Android)

### **Option A - Installation instantanée (recommandé)**

1. **Ouvrez ce lien sur votre téléphone Android :**
   - Cherchez un service d'hébergement gratuit de fichiers statiques
   - OU utilisez GitHub Pages (gratis et simple)

2. **Une fois sur le site :**
   - Attendez que la page charge complètement
   - Cliquez sur le menu du navigateur (3 points en haut)
   - Cliquez sur **"Ajouter à l'écran d'accueil"**
   - L'app apparaît maintenant comme une vraie app ! 📱

3. **Utilisez l'app :**
   - Les données se sauvegardent automatiquement
   - Fonctionne sans Internet
   - Vos données restent sur VOTRE téléphone uniquement

---

## 📱 Fonctionnalités

### 📍 Marches
- Enregistrez chaque marche (heure, durée, pas)
- Voir le résumé quotidien automatique
- Supprimer une marche si besoin

### 😔 Douleur
- Notez votre niveau de douleur chaque jour (0-10)
- Visualisez la semaine avec des barres de couleur
- Vert (0-3) → Jaune (4-6) → Rouge (7-10)

### 📈 Graphique
- Voyez vos 4 dernières semaines d'un coup d'œil
- Rouge = douleur moyenne
- Bleu = nombre de pas
- Voyez votre progression !

---

## 💾 Où sont vos données ?

✅ **100% sur votre téléphone**
✅ **Aucun serveur, aucun cloud**
✅ **Complètement privé**
✅ **Fonctionne hors ligne**

Les données se sauvegardent automatiquement chaque fois que vous ajoutez une marche ou une note de douleur.

## 🔧 Si vous avez un ordinateur (hébergement)

Vous pouvez héberger cette app **gratuitement** sur :

### Option 1 : GitHub Pages (5 min)
1. Créez un compte GitHub
2. Créez un repo "suivi-guerison"
3. Uploadez les 3 fichiers : `index.html`, `manifest.json`, `sw.js`
4. Activez GitHub Pages dans les settings
5. Votre app est en ligne ! 🎉

### Option 2 : Netlify (3 min)
1. Allez sur https://www.netlify.com
2. "Drag and drop" les 3 fichiers
3. C'est live !
